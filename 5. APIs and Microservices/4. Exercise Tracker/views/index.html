Exercise Tracker Microservice

A freeCodeCamp Javascript Back-End Challange

This challange makes use of NodeJS, ExpressJS and MongoDB with Mongoose.

Challange:

  • I can create a user with the POST method from a form to /api/exercise/new-user and will return an object with the username and unique _id.
  • I can add an exercise to any user with the POSTmethod from a form to /api/exercise/add and will then return the user object with the exercise fields added.
  • I can return an array of all users at the endpoint api/exercise/users.
  • I can GET a full exercise log of any user at /api/exercise/log? with a parameter of the username or the unique _id.

Create a New User

POST /api/exercise/new-user


Add Exercises to an Existing User

POST /api/exercise/add

GET Usage Examples:

GET /api/exercise/log?{userId}[&from][&to][&limit]
Click me: http://localhost:3000/api/exercise/users GET All Users
Click me: http://localhost:3000/api/exercise/log?userId={userId} GET User Log
Click me: http://localhost:3000/api/exercise/log?userId=nazarja&from=2011-02-01&to=2011-02-02limit=2 GET User Log
Click me: http://localhost:3000/api/exercise/log?userId=nazarja&from=2011-02-02&to=2011-02-03limit=2 GET User Log

{ } = required, [ ] = optional

from, to = dates (yyyy-mm-dd); limit = number
