Unix Timestamp Microservice

A freeCodeCamp Javascript Back-End Challange

This challange makes use of NodeJS and ExpressJS

Challange:

  • Given either a Unix (Epoch) Timestamp or a natural date within the url of this page.
  • A JSON object is returned with both the unix timestamp and the natural language form of that date.
  • I f an Invalid date is entered the JSON object will return null.

Usage:

In the url bar you can enter either a natural language date or a unix timestamp:

Examples:

Click me: http://localhost:8080/April%2015,%202017
Click me: http://localhost:8080/December 10, 2145
Click me: http://localhost:8080/3 Feb, 1988
Click me: http://localhost:8080/1492210800
Click me: http://localhost:8080/5552223333
Click me: http://localhost:8080/570844800

Output:

{ "unix": 1450137600, "natural": "April 15, 2017" }
{ "unix": null, "natural": null }
