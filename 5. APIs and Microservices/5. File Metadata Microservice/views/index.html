File Metadata Microservice

A freeCodeCamp Javascript Back-End Challange

This challange makes use of NodeJS , ExpressJS and the npm multer module.

Challange:

  • To be able to submit a FormData object that includes a file upload.
  • When a file is uploaded, A JSON response with the file size in bytes will be returned.

Usage:

You will need to select a file from your computer or device in order for the file metadata to be read.

Output:

{ "File Size MB": 14 }
